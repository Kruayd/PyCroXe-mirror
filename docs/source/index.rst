===================
Welcome to PyCroXe!
===================

The Python API that let you import data from
`CroXe <https://codeberg.org/Kruayd/CroXe>`_, with intuitive calls and minimal
effort.

If you are looking for the database itself, visit:
`https://codeberg.org/Kruayd/CroXe <https://codeberg.org/Kruayd/CroXe>`_.
